createTask

Эта мутация добавляет задачу к дню.

Input fields

body (String!)

Передается описание задачи.

day_id (Int!)

Передается ID дня.

is_published (Boolean)

Передатеся значение "Опубликована ли задача?"

name (String)

Передается наименование задачи.

number_of_percentages (Float!)

Передается количество процентов, которое дается за выполнение задачи в дне. Мы должны дойти до ста процентов, тогда день будет считаться сделанныым и больше в него нельзя будет добавлять задач.

number_of_points (Int!)

Передается колчиество очков, которое мы даем пользователю за выполнение данной задачи. Нужно в скиллах польхзователя.

serial_number (Int!)

Порядковый номер задачи в дне.

skills ([String])

Добавляет и создаёт скиллы к задаче.

Return fields

body (String!)

Тело задания. Может быть в HTML.

day (Day)

Связь с объектом дня.

day_id (Int!)

ID дня.

id (Int!)
name (String!)

Наименование задания.

number_of_percentages (Int!)

Проценты, начисляемые за выполнение задания. Когда выполняешь все задания в объекте UserDay становится 100% и можно сказать, что пользователь прошел все таски за день.

number_of_points (Int!)

Количтесво очков, начиляемое пользователю за выполнение задания.

serial_number (Int!)

Порядковый номер задания по прохождению на день.

skills ([Skill]!)

Навыки прибретаемые в процессе выполнения задания.
