UserTask

Объект, обозначающий, что пользователь проходит задание.

Fields

decisions ([Decision]!)

Решения, закрепленные за данным объектом.

id (Int!)
intervals ([UserTaskInterval]!)

Промежутки выполнения пользователем задания.

is_completed (Boolean!)

Значение, обозначающее выполнил ли пользователь задание.

is_current (Boolean!)

Значение, обозначающее выполняет ли пользователь задание в данный момент.

last_decision (Decision)

Объект последнего решения, который пользователь отправил, решая задание.

related_user_tasks ([UserTask]!)

Остальные задачи, который выполнял пользователь, относящиеся к текущему дню.

task (Task)

Связь с объектом задачи.

task_id (Int!)

ID задачи.

user_day (UserDay)

Связь с объектом UserDay.

user_day_id (Int!)

ID объекта UserDay.
