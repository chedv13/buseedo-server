Decision

Решение, которое пользователь закрепил за задачей.

Fields

body (String!)

Тело решения задачи. Может быть в HTML.

feedback (String)

Фидбек преподавателя. Может быть в HTML.

id (Int!)
status (String!)

Статус решения. Можем быть: ["pending", "failed", "verified"]. 'pending' - решение ещё не проверено преподавателем. 'verified' - преподаватель счёл решение правильным. 'failed' - преподаватель счёл решение неправильным.

teacher (User)

Связь с объектом преподавателя курса.

user_id (Int!)

ID преподавателя курса.
