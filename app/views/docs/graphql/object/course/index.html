Course

Объект курса.

Fields

background_image_url (String!)

Это поле предназначено для того, чтобы в ios-версии (и возможно где-то еще) вставлять изображение позади описания курса.

Argument Type Description
style String

Possible styles: {"common_60"=>"60x60>"}. Arguments are keys of hash.
course_teachers ([CourseTeacher]!)

Здесь мы получаем связь course_teachers, через которую можем достучаться до преподавателей курса.

cover_url (String!)

Иконка курса.

Argument Type Description
style String

Possible styles: {"common_60"=>"60x60>"}. Arguments are keys of hash.
days (DayConnectionWithTotalCount)

Связь дней с курсом.

Argument Type Description
after String

Returns the elements in the list that come after the specified cursor.
before String

Returns the elements in the list that come before the specified cursor.
first Int

Returns the first n elements from the list.
last Int

Returns the last n elements from the list.
description (String)

Краткое текстовое описание курса. Нужно для того, чтобы первично ознакомиться с курсом.

final_number_of_points (Int!)

Количество баллов, котороые можно получить за прохождение данного курса (в общем, нужно в скиллах и для определениия левела пользователя в будущем).

full_description (String)

Полное текстовое описание курса. Возможно в HTML.

id (Int!)
